# 熹茗 AI 加盟商赋能平台 · Demo

针对熹茗 (Ximing) 在新品研发后向加盟商发布过程中暴露的三大痛点 —— **知识更新难、加盟商学习成本高、运营难度大** —— 提供一套「业务+技术」端到端的 AI 降本增效落地方案与可交互演示。

## 🌟 在线演示

> 部署后访问：**[https://loadingtz.github.io/ximing-ai-demo/](https://loadingtz.github.io/ximing-ai-demo/)**

## 📦 包含内容

| 模块 | 说明 |
|------|------|
| ① 新品 AI 助手 | 基于 RAG 的自然语言问答，加盟商问什么答什么 |
| ② 新品知识库 | 总部一次发布，AI 自动生成 SOP / 话术 / 营养表 / 常见问答 |
| ③ 智能培训 | 个性化出题，连答连讲，培训时长 -70% |
| ④ 运营驾驶舱 | AI 直接给「具体建议」：补什么、推什么、调什么班 |
| ⑤ 视觉质检 | 出品照片上传，AI 评分色泽 / 分量 / 装饰 / 温度 |
| ⑥ 架构与流程 | 业务痛点 ↔ AI 技术映射、技术团队交付清单 |
